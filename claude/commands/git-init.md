既存プロジェクトにgit管理を導入する。以下の手順で進める：

1. 現在のディレクトリにすでに `.git` が存在するか確認する。存在する場合は中断して報告する。
2. プロジェクトの内容を確認し、使用している言語・フレームワークを特定する。
3. ユーザーに以下を確認する：
   - GitHubリポジトリ名（デフォルトはディレクトリ名）
   - 公開設定（public / private）
4. 確認後、以下を実行する：
   - `gibo dump` で特定した言語・フレームワークに合った .gitignore を生成する（例: `gibo dump Node > .gitignore`）
   - 生成された .gitignore の内容をユーザーに提示し、追加が必要か確認する
   - `git init`
   - 初回コミットを作成
   - `gh repo create` でGitHubリポジトリを作成しプッシュする
